宝塔面板安装记录

购买了一个轻量服务器，像我这种对Linux不熟悉的还是安装一个图形化管理面板好管理一点，运营服务器使用宝塔面板图形化管理。

使用ssh连接服务器

可以使用网页的SSH连接服务器

或者使用软件进行连接（如Xshenll 7），在命令行输入从宝塔Linux安装页面复制的安装脚本命令

yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh

在 Do you want to install Bt-Panel to the /www directory now?(y/n): 按键盘 y

安装完成后会给一个面板地址，需要在云服务器管理控制台打开相应的端口 8888 进入控制台找到防火墙/安全组 添加一个TCP 端口为8888 的端口

完成后打开命令行给的外网地址,外网面板地址是类似这样的: http://xx.xx.xx.xx.:8888/xxxxxx，打开后进去后就是面板主页了，

为了服务器安全！！！ 切记要修改 面板端口，安全入口，面板用户，面板密码，

给面板加上域名解析

进入域名DNS解析管理，添加一个记录类型为 A 记录值 xxxx.xxxx.xxxx.xxxx (服务器ipV4的地址)
